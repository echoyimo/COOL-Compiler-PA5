# COOL代码生成器 - 编译原理PA5作业

## 项目概述
本项目实现了COOL(Classroom Object-Oriented Language)语言的代码生成器，将语义分析后的AST转换为可运行的MIPS汇编代码。

## 项目特性
- ✅ AST到MIPS汇编的完整转换
- ✅ 支持基本表达式、控制结构、类与方法
- ✅ 动态分发和静态分发机制
- ✅ 完整的继承处理
- ✅ 在SPIM模拟器上验证通过
